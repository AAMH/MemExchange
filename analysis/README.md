# Analysis

This directory contains the R scripts used to analyze experimental results and generate the figures and statistical analyses presented in the MemExchange paper and my PhD dissertation.

The scripts operate on the raw benchmark logs collected during experiments and produce publication-ready figures, summary statistics, and significance tests.

## Directory Contents

Typical scripts include:

- Figure generation for paper and dissertation plots
- Latency and throughput analysis
- Cache hit and miss rate analysis
- Memory utilization analysis
- Statistical significance tests
- Auxiliary scripts for processing experiment logs

## Workflow

The typical evaluation workflow is:

```text
Run experiments
        ↓
Collect benchmark logs
        ↓
Process logs
        ↓
Run R scripts
        ↓
Generate figures and statistics
```

## Reproducibility

The goal of this directory is to make the evaluation as reproducible as possible. Every figure in the paper was generated from raw experimental data using scripts contained in this directory.

Depending on the experiment, some scripts expect input files generated by the automation scripts in `scripts/experiments/`.

If you organize experimental logs differently, you may need to update the input paths accordingly.

## Requirements

Most scripts require:

- R (version 4.x or newer recommended)
- Common packages such as:
  - ggplot2
  - dplyr
  - tidyr
  - readr
  - reshape2
  - gridExtra

Additional package dependencies are documented at the beginning of each script.

## Notes

Some scripts were written specifically for generating figures used in the paper and dissertation. While they are fully functional, they assume the naming conventions and directory structure used during the evaluation.
